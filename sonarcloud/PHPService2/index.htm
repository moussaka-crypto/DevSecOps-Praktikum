XSSeducation is a set of XSS vulnerable PHP pages for testing (and fun, don't forget the fun).

Level 1

  • Basic XSS - execute code in the page
  • JavaScript XSS - execute code in the page

Level 2

  • Filtered XSS - alert() the credit card number

Level 3

  • Chained XSS - change the password to 1234, the user must NOT notice this behavior

Files

These are files you can embed in the page if you wish. Not very well documented, however.
